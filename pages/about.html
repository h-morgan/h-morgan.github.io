About Me

Thanks for visiting my site! I’m a developer based out of Boston, MA. I went to Worcester Polytechnic Institute (WPI) in Worcester, MA - where I graduated with a degree in engineering in 2016. I worked as a civil engineer for a little while right after college, but soon realized that it was not my passion. I started taking data science courses online after work through a recommendation from a friend of mine, which introduced me to the Python programming language. After that I was hooked - I loved the way the language was like a puzzle that needed to be figured out in order to make it do what you wanted. I wanted to learn more - about the syntax, the different data structures, functions, and ultimately how to use programming to create anything I wanted.

After that I knew I wasn’t really hooked on data science - I was more hooked on learning the language commonly used for data science. This was when I realized my true pull was towards learning languages used to develop things - which led me to pursue the frontend languages of the web: HTML, CSS, and JavaScript. I loved the way HTML and CSS could be used to so quickly to set up simple websites - and how immediately the results could be seen. JavaScript came next, and I was eventually learning about adding app-like functionality to websites. I then became curious about how the backend of a website could support this app-like functionality to make even more complex applications. I learned about how databases could be incorporated into the backend of an app and to have specific bits of information be served to the client as requested.

I continue to expand my developer knowledge base every day - creating things in my spare time after work and on weekends. You can check out some of the things I’ve worked on via my Github or the Projects page of this site. Feel free to get in touch with me via the Contact Me section of this site!
